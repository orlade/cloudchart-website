CloudChart takes the pain out of your cloud

AWS is a powerful platform, but administration can be a real headache.

CloudChart gives you the user experience you deserve for monitoring and managing your AWS infrastructure.

Using the AWS API, your resources are not just visualised, but common tasks can be performed in a single click.


Warning! CloudChart is still heavily under development, and production use is not recommended.

On the other hand, CloudChart is open source so if you like it, lend a hand!
